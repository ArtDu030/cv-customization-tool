========================================
       CV CUSTOMIZATION TOOL
           User Guide
========================================

QUICK START:
1. Double-click "Start-CV-Tool.bat" to start the application
2. Wait for it to install dependencies and build (first time only)
3. Your web browser will open automatically at http://localhost:3000 or http://192.168.1.88:3000/
4. Use the application to customize your CV based on job postings

REQUIREMENTS:
- Windows PC
- Node.js (will be installed automatically if missing)
- Internet connection (for CV customization API)

HOW TO USE:
1. Select your target language (English or German)
2. Upload a job posting text file or paste the job description
3. Enter your Claude API key (get it from https://console.anthropic.com)
4. Click "Customize CV" to generate a tailored version
5. Review the customized CV
6. Export to PDF when satisfied

TROUBLESHOOTING:
- If port 3000 is busy, the app will use port 3001 or 3002
- Make sure you have a stable internet connection
- If the build fails, delete the .next folder and try again
- Close the command window to stop the application

TECHNICAL NOTES:
- This app runs locally on your computer
- No data is sent to external servers (except Claude API for CV customization)
- All processing happens on your machine

SUPPORT:
- Check that Node.js is properly installed
- Ensure no antivirus is blocking the application
- Make sure you have sufficient disk space for dependencies
